修复详情

1. ISSUE-001 修复方案：
   - 移除 Lombok 注解，手动实现数据模型类的 getter/setter 和 Builder 模式
   - 使用 -proc:none 参数禁用注解处理

2. ISSUE-002 修复方案：
   - 在 OkHttpClient 中添加 SSL 证书信任配置
   - 实现信任所有证书的 TrustManager 和 HostnameVerifier

3. ISSUE-003 修复方案：
   - 延长 OkHttpClient 和 Jsoup 的超时时间
   - 使用 CompletableFuture 设置 3 秒超时

4. ISSUE-004 降级方案：
   - 实现模拟数据降级策略
   - 当真实数据源失败时返回预设的模拟价格

5. ISSUE-005 修复方案：
   - 创建批处理版本脚本（.bat）替代 PowerShell 脚本（.ps1）
   - 使用 cmd /c 执行命令

6. ISSUE-006 修复方案：
   - 使用 compareTo() 方法代替 equals() 进行 BigDecimal 比较
   - 使用宽松的范围断言验证数值
